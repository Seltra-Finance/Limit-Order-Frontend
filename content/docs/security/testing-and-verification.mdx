### Reproduce locally

```bash
cd contracts
forge test
FOUNDRY_PROFILE=ci forge test

RUN_MAINNET_FORKS=1 AVAX_RPC_URL=https://api.avax.network/ext/bc/C/rpc forge test --match-path 'test/fork/*.t.sol'

cd ../services
npm ci
npm test
npm run typecheck
npm run build
```
